## OAuth Authentication in Next.js
Приложение на **Next.js** с поддержкой авторизации через **Google**, **Yandex** и стандартную форму входа по **email и паролю**. 
## Technologies Used
- React 19
- Next 15
- Tailwind CSS
- React Hook Form
- React Query
- TypeScript
- Axios
- Framer Motion
- React Hot Toast
- Google reCAPTCHA
- Turbopack

## Getting Started
First, run the development server:

```bash
npm run dev
# or
yarn dev
```
